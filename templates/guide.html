Guide How to Use:

How to Connect on RPB:

    Follow the instructions below:

  1. Make sure your device and the Raspberry Pi are connected on the same network.

  2. Find the Raspberry Pi IP address on your network.

  3. Input Raspberry Pi IP address on your browser url address bar.

  4. Make sure the IP address is correct and the port must be 5000.(Example input: 172.0.0.0:5000)

  5. If all inputs are correct you will be redirect to home windows of RPB application.

  6. When you encounter some problem try to reboot the Raspberry Pi.


Inside the Application

    The work arround guide on how to use the application.

  • On Menu navigation you can access the Home, Login/Logout, About, Contact, Settings on the upper part, and the credits developer of the project list on lower part. You can click the names to view there social media.

  • In the home tab eplained what the application is for.

  • Login page and logout page, if you want login as Administrator to edit the settings information and logout to exit as Administrator.

  • About tab emphasize the information about the application.

  • Contact tab list the contact inforation of the developers.

  • Settings tab can only access if you are login as Administrator. You can edit the Stablishment name, Administrator registered name, and Phone number of the recipient for SMS message.

    • To edit the settings make sure the Recycle Pi Bin Status is not running.
    • Double check the recipient number inputs, the number registered is where the message of warning if RPB trash can is already full will be send.
    • Stablishments name is where the RPB located and Administrator name is the Administrator that will monitor the application
    • Edit password or forgot password can be found on settings if you want to change the password registered.
    • Make sure to logout if you are done editing the information.

  • On the bottom part of the screen locate the Home, Statistics, Guide, Settings and About tabs.

    • Home same function of the Home tab mention above.
    • Statistics tab you can see the status of RPB if trash can is already full, if it is running or not, History Information on when last emptied the trash can and the last warning message sent when the trash can is full and the summary of information registered to the RPB application.
    • Guide tab you can read the guide on how to use RPB and detailed of the application.
    • Settings tab is also same content of the settings Tab mention above.
    • About tab same content of above mention.

When Reboot or Shutdown the RPB

    To shutdown or reboot the RPB for trobleshooting please go to Settings and you will see the Reboot and Shutdown buttons their.

  1. If you want to reboot the RPB, click Reboot button, the RPB now will shutdown and wait for about 3-5 minutes to turn on itself again.

    Refresh the webpage of your browser to see if its on again. If the page loaded perfectly the RPB is ready to use again. (You may need to login again for settings to update).

  2. When you shutdown the RPB on Settings tab button, you need to manually turn in On again.

    To Turn On the RPB manually you need to unplug and replug the RPB power supply.
